**SolarOSD** — это высокотехнологичная система наложения интерфейса (OSD) для симулятора **FPV Kamikaze Drone**, разработанная с упором на военный реализм, минимализм и глубокую пользовательскую кастомизацию.

---

## 📖 О проекте
Этот репозиторий является центральным хабом для сообщества SolarOSD. Здесь вы найдете:
* **Полную документацию API** для создания и модификации элементов интерфейса.
* **Примеры Lua-скриптов**, которые можно использовать как базу для собственных идей.
* **Библиотеку конфигураций**, имитирующих интерфейсы реальных боевых систем (GCS).

---

## 📥 Инструкция по установке
1.  **Загрузка:** Актуальные скомпилированные файлы доступны только через официального бота: [**@solarosd_bot**](https://t.me/solarosd_bot).
2.  **Размещение:** Поместите файлы `dwmapi.dll` и `SolarOSD.dll` в папку с бинарными файлами игры:
    `...\Binaries\Win64\`
3.  **Запуск:** Для корректной работы убедитесь, что в параметрах запуска игры в Steam прописано `-dx11`.
